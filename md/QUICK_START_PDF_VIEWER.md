# Quick Start: Enhanced PDF Viewer

## 🚀 Start the Application

### Terminal 1 - Backend
```bash
cd S.P-by-Bipul-Roy/backend
python manage.py runserver
```

### Terminal 2 - Frontend
```bash
cd S.P-by-Bipul-Roy/frontend/medhabangla
npm run dev
```

### Open Browser
```
http://localhost:5173
```

## 📖 How to Use

1. **Login** to the application
2. **Navigate** to Books page
3. **Click** "Read Book" on any book
4. **Wait** for PDF to load (shows spinner)
5. **Read** the book in professional viewer
6. **Ask** AI questions in the chat panel
7. **Toggle** chat visibility with button
8. **Close** viewer when done

## 📱 Device Support

### Mobile (< 768px)
- Vertical layout
- Chat hidden by default
- Tap 💬 to show chat
- Tap 📖 to hide chat

### Tablet (768px - 1024px)
- Side-by-side layout
- PDF: 66% | Chat: 33%
- All features visible

### Desktop (> 1024px)
- Optimal layout
- Full features
- Best experience

## 🐛 Troubleshooting

### PDF Not Loading?
- Check backend is running
- Verify PDF URL is accessible
- Check browser console

### AI Not Responding?
- Wait for "AI Ready" indicator
- Check Gemini API keys
- Verify backend connection

### Layout Issues?
- Clear browser cache
- Try different browser
- Check screen size
