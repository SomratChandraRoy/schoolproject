## Root Cause Analysis

### Why These Errors Occurred

1. **ElevenLabs UI Components**
   - These are official ElevenLabs UI components
   - Designed for Next.js by default (hence "use client")
   - Require specific ElevenLabs packages

2. **Transitive Dependencies**
   - `@elevenlabs/elevenlabs-js` was installed via `@elevenlabs/cli`
   - Not explicitly listed in dependencies
   - Could cause issues if CLI package is removed

3. **Framework Mismatch**
   - Components use Next.js conventions
   - Project uses Vite + React
   - "use client" directive is ignored but unnecessary

## Component Overview

### speech-input.tsx
**Purpose**: Speech-to-text input component
**Features**:
- Microphone recording
- Real-time transcription
- Partial and committed transcripts
- VAD (Voice Activity Detection)
- Multiple audio formats

**Key Exports**:
- `SpeechInput` - Main container
- `SpeechInputRecordButton` - Record/stop button

- Time labels

**Key Exports**:

- `TranscriptViewerWords` - Word display

## Known Limitations

### 1. API Token Required
- Components need ElevenLabs API token
- Must be provided via `getToken` prop
- Token should be fetched from backend

### 2. Audio Alignment Data
- Transcript viewer needs alignment data
- Format: `CharacterAlignmentResponseModel`
- Must be generated by ElevenLabs API

### 3. Browser Permissions
- Speech input requires microphone permission
- User must grant permission
- Handle permission denied errors

## Usage Example

### Speech Input
```tsx
import {
  SpeechInput,
  SpeechInputRecordButton,
  SpeechInputPreview,
  SpeechInputCancelButton,
} from '@/components/ui/speech-input';

function MyComponent() {
  const getToken = async () => {
    const response = await fetch('/api/elevenlabs/token');
    const { token } = await response.json();
    return token;
  };

  return (
    <SpeechInput
      getToken={getToken}
      onChange={(data) => console.log(data.transcript)}
    >
      <SpeechInputRecordButton />
      <SpeechInputPreview />
      <SpeechInputCancelButton />
    </SpeechInput>
  );
}
```
