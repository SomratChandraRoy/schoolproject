🚀 SETUP (ONE TIME)
───────────────────────────────────────────────────────────────────────

Step 1: Run Migrations
-----------------------
cd S.P-by-Bipul-Roy/backend
python manage.py makemigrations ai
python manage.py migrate

Step 2: Start Servers
----------------------
# Terminal 1: Backend
cd S.P-by-Bipul-Roy/backend
python manage.py runserver

# Terminal 2: Frontend
cd S.P-by-Bipul-Roy/frontend/medhabangla
npm run dev

───────────────────────────────────────────────────────────────────────
🎯 HOW TO USE
───────────────────────────────────────────────────────────────────────

1. Login as Admin
2. Go to: http://localhost:5173/superuser
3. Click "🤖 AI Settings" button
4. Select AI Provider:
   - Gemini Only (fast, has limits)
   - Ollama Only (unlimited, needs EC2)
   - Auto (tries Gemini, falls back to Ollama) ← RECOMMENDED
5. Click "Test Connection" to verify
6. Click "Save Settings"
7. Done! All AI features now use your selected provider

───────────────────────────────────────────────────────────────────────
🎨 THREE MODES
───────────────────────────────────────────────────────────────────────

MODE 1: Gemini Only 🌟
-----------------------
✅ All AI features use Gemini API
✅ Fast and reliable
⚠️ Has quota limits (160/day)
📌 Use when: Ollama server is down

MODE 2: Ollama Only 🦙
-----------------------
✅ All AI features use AWS Ollama
✅ Unlimited requests
✅ No API costs
⚠️ Requires EC2 running
📌 Use when: Want maximum cost savings

MODE 3: Auto (Recommended) ⚡
-----------------------------
✅ Tries Gemini first
✅ Falls back to Ollama if Gemini fails
✅ Best reliability
✅ Optimal cost/performance
📌 Use when: Production environment
