================================================================================
                    AWS Ollama Setup - Quick Start (বাংলা)
================================================================================

🎯 লক্ষ্য: AWS EC2-তে Ollama install করে যেকোনো জায়গা থেকে ব্যবহার করা

================================================================================

📋 যা যা লাগবে:
-----------------
✓ AWS Free Trial Account
✓ SSH Client (Windows-এ built-in আছে)
✓ 30 minutes সময়

================================================================================

🚀 Quick Setup (5 Steps):
--------------------------

STEP 1: AWS EC2 Instance তৈরি করুন
------------------------------------
1. AWS Console → EC2 → Launch Instance
2. Name: ollama-server
3. AMI: Ubuntu 22.04 LTS
4. Instance type: t2.medium (⚠️ t2.micro যথেষ্ট নয়!)
5. Key pair: Create new → Download করুন
6. Security Group:
   - SSH (port 22) - My IP
   - Custom TCP (port 11434) - Anywhere
7. Storage: 30GB
8. Launch!

STEP 2: SSH Connect করুন
--------------------------
# Windows PowerShell:
ssh -i "your-key.pem" ubuntu@YOUR_EC2_IP

# Example:
ssh -i "ollama-key.pem" ubuntu@54.123.45.67

STEP 3: Ollama Install করুন
-----------------------------
# Instance-এ এই commands run করুন:

# System update
sudo apt update && sudo apt upgrade -y

# Ollama install
curl -fsSL https://ollama.com/install.sh | sh

# External access configure
sudo mkdir -p /etc/systemd/system/ollama.service.d
sudo tee /etc/systemd/system/ollama.service.d/override.conf > /dev/null <<EOF
[Service]
Environment="OLLAMA_HOST=0.0.0.0:11434"
EOF

# Service restart
sudo systemctl daemon-reload
sudo systemctl restart ollama

# Model download
ollama pull llama3.2

STEP 4: Test করুন
------------------
# আপনার local computer থেকে:
curl http://YOUR_EC2_IP:11434/api/generate -d "{\"model\":\"llama3.2\",\"prompt\":\"Hello\",\"stream\":false}"

# যদি response আসে = Success! ✅

STEP 5: React App Setup করুন
------------------------------
# 1. .env file তৈরি করুন:
cd S.P-by-Bipul-Roy/frontend/medhabangla
copy .env.example .env

# 2. .env file edit করুন:
REACT_APP_OLLAMA_URL=http://YOUR_EC2_IP:11434

# 3. Dev server চালু করুন:
npm run dev

# 4. Browser-এ যান:
http://localhost:5173/ollama

# 5. AI-র সাথে chat করুন! 🎉

💰 Cost Saving Tips:
--------------------
1. Instance stop করুন যখন ব্যবহার করছেন না
2. Elastic IP use করুন (IP same থাকবে)
3. CloudWatch alarm set করুন billing-র জন্য
4. Free tier limit: 750 hours/month

================================================================================

🔧 Common Issues:
-----------------

Problem: Connection refused
Solution: 
  - Check: sudo systemctl status ollama
  - Check: Security Group port 11434 open আছে কিনা

Problem: Model not found
Solution:
  - Run: ollama list
  - Run: ollama pull llama3.2

Problem: Out of memory
Solution:
  - Use smaller model: ollama pull gemma2:2b
  - Or upgrade to t2.large
