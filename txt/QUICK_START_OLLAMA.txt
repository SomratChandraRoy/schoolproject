⚠️ TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────

PROBLEM: Connection timeout
SOLUTION: 
  1. Check if EC2 instance is running
  2. Check Security Group allows port 80
  3. SSH and check: sudo systemctl status ollama

PROBLEM: 401 Unauthorized
SOLUTION:
  1. Verify credentials in .env file
  2. Test with: curl -u bipul:Bipul\$Ollama\$Roy\$2026\$ http://16.171.19.161/api/tags

PROBLEM: Slow responses
SOLUTION:
  1. Normal for first request (model loading)
  2. Subsequent requests should be faster
  3. Consider upgrading EC2 instance
